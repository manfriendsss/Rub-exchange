# Đổi Tiền - Ứng dụng quy đổi tiền tệ hiện đại

Chào mừng bạn đến với **Đổi Tiền**, một ứng dụng quy đổi tiền tệ tinh gọn, mạnh mẽ và mượt mà, được tối ưu hóa cho trải nghiệm di động.

## 🌟 Chức năng chính

- **Quy đổi đa năng**: Hỗ trợ hơn 15 loại tiền tệ phổ biến nhất thế giới (USD, EUR, CNY, JPY, KRW, RUB, THB, LAK, SGD, GBP, AUD, v.v.).
- **Tỷ giá trực tuyến**: Tự động cập nhật tỷ giá hối đoái thực tế từ API toàn cầu.
- **Tùy chỉnh linh hoạt**: 
  - Thay đổi cả tiền gốc và tiền đích một cách dễ dàng.
  - Chạm vào tỷ giá để tự nhập tỷ giá mong muốn.
  - Chuyển đổi ngược lại chỉ với một chạm.
- **Bàn phím thông minh**: Bàn phím ảo tích hợp khả năng tính toán cộng, trừ, nhân, chia trực tiếp khi nhập liệu.
- **Trải nghiệm PWA**: Có thể cài đặt trực tiếp vào màn hình chính điện thoại (Android/iOS) để sử dụng như một ứng dụng gốc.
- **Giao diện hiện đại**: Thiết kế Dark Mode chuyên nghiệp với các hiệu ứng chuyển động mượt mà bằng `framer-motion`.
- **Hướng dẫn sử dụng**: Hệ thống Tutorial tích hợp cho người mới bắt đầu.
- **Tiện ích sao chép**: Chạm vào kết quả để sao chép số tiền đã quy đổi vào bộ nhớ tạm.
- **Lưu trữ cá nhân**: Tự động ghi nhớ loại tiền bạn chọn và tỷ giá tùy chỉnh của bạn cho lần sử dụng sau.

## 🛠 Công nghệ sử dụng

- **Frontend**: React 18 + Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion (motion/react)
- **Icons**: Lucide React
- **API**: ExchangeRate-API (cho tỷ giá trực tuyến)
- **Deployment**: Tương thích hoàn toàn với iFrame và PWA

## 📱 Cài đặt

Ứng dụng hỗ trợ tốt nhất khi được cài đặt vào màn hình chính:
1. Mở ứng dụng trong trình duyệt điện thoại.
2. Với iOS (Safari): Nhấn "Chia sẻ" > "Thêm vào MH chính".
3. Với Android (Chrome): Nhấn nút "Cài đặt ngay" trong popup hoặc menu trình duyệt.
